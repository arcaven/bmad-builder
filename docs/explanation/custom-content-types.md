---

BMad supports several categories of custom content.

## Module Types

| Type | Description |
|------|-------------|
| **Stand-Alone** | Independent systems |
| **Add-On** | Extensions that enhance another module |
| **Global** | Cross-cutting functionality for all content |

## Custom Agents

Custom agents can be:
- Used within custom modules
- Standalone tools
- Integrated with existing workflows

## Custom Workflows

Workflows range from simple single-file prompts to multi-step processes:
- User facilitation and engagement
- Business processes
- Content generation
- Utility functions

## Building

Use **BMad Builder** workflows for guided creation:

| Task | Command |
|------|---------|
| Create agent | `bmad-bmb-create-agent` |
| Create workflow | `bmad-bmb-create-workflow` |
| Create module | `bmad-bmb-create-module` |
